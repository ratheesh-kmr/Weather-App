About Forecaster

Welcome to Forecaster! This application provides accurate and up-to-date weather information for cities using predefined values. It was created by Ratheesh Kumar using the Angular framework. Simply enter the name of a city, and you'll get the latest weather data including temperature, high and low forecasts, and a brief description of the current conditions.

Our goal is to make weather information easily accessible and understandable for everyone. Whether you're planning a trip, preparing for your day, or just curious about the weather, Forecaster has you covered.

Features

  • Predefined weather data for selected cities
  • High and low temperature forecasts
  • Weather descriptions
  • Built with Angular framework

About Us

Forecaster was developed by Ratheesh Kumar, a dedicated individual passionate about providing accurate and reliable weather information. By using Angular and predefined data, we aim to provide a seamless and informative weather experience.
